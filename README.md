# דשבורד פיננסי משפחתי

## הפעלה ראשונה

### אפשרות א׳ — קליק כפול (הכי פשוט)
1. לחץ לחיצה ימנית על הקובץ `הפעל.command`
2. בחר "פתח" → "פתח" שוב בחלון האבטחה
3. הדפדפן יפתח אוטומטית לכתובת http://localhost:5001

(בפעם הראשונה בלבד תצטרך לאשר את הקובץ ב-Security settings)

### אפשרות ב׳ — דרך הטרמינל
```bash
cd /path/to/family_finance
pip3 install flask
python3 app.py
```
ואז פתח את הדפדפן ב: http://localhost:5001

---

## שימוש יומיומי

1. פתח את `הפעל.command` (קליק כפול)
2. הדפדפן נפתח אוטומטית
3. עדכן יתרות, הוסף/ערוך פוליסות
4. סגור את חלון הטרמינל כשסיימת
